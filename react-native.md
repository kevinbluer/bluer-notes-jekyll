#### Running on iOS

> react-native start

To run your app on iOS:
   cd /Users/bluer/Developer/AwesomeProject
   react-native run-ios
   - or -
   Open /Users/bluer/Developer/AwesomeProject/ios/AwesomeProject.xcodeproj in Xcode
   Hit the Run button
